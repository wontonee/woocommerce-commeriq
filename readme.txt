== Installation ==

= Automatic Installation =

1. Log in to your WordPress dashboard
2. Navigate to Plugins > Add New
3. Search for "CommerIQ"
4. Click "Install Now" and then "Activate"
5. Go to WooCommerce > CommerIQ to configure your license

= Manual Installation =

1. Download the plugin ZIP file
2. Log in to your WordPress dashboard
3. Navigate to Plugins > Add New > Upload Plugin
4. Choose the ZIP file and click "Install Now"
5. Activate the plugin
6. Go to WooCommerce > CommerIQ to configure your license

= Configuration =

1. After activation, navigate to WooCommerce > CommerIQ
2. Go to the "License" tab
3. Enter your license key and domain
4. Click "Activate License"
5. Start using AI-powered features!

== Frequently Asked Questions ==

= Do I need a license to use CommerIQ? =

Yes, CommerIQ requires a valid license key to access all features. You can purchase a license at [wontonee.com/commeriq](https://wontonee.com/commeriq)

= How do I generate AI content for my products? =

1. Go to any product edit page
2. Find the "AI Content" button above the product description editor
3. Click the button to generate compelling product descriptions
4. Review and save your content

= Can I generate images for existing products? =

Yes! On any product edit page, click the "Generate AI Image" button in the Product Image section. The AI will create a professional image based on your product name and description.

= How does price comparison work? =

Navigate to the Price Comparison section in the product data area and click "Run Comparison". CommerIQ will analyze prices across multiple platforms and provide competitive insights.

= Is my data secure? =

Absolutely! We use industry-standard encryption and security practices. Your data is transmitted securely and stored safely. We never share your information with third parties.

= What happens if my license expires? =

If your license expires, AI features will be disabled until you renew. Your existing content and images remain intact. Renew your license to continue using all features.

= Can I use CommerIQ on multiple sites? =

License terms vary by plan. Check your license details at [wontonee.com/commeriq](https://wontonee.com/commeriq) for multi-site availability.

= How often is the plugin updated? =

We regularly release updates with new features, improvements, and bug fixes. Updates are automatic through your WordPress dashboard.

= Do you offer support? =

Yes! Licensed users receive priority support. Contact us through our [Support Center](https://wontonee.com/support) for assistance.
